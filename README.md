# API Testing Project

To repozytorium zawiera moją kolekcję testów API stworzoną w Postmanie.

## Co zawierają testy?
* **Vanilla JavaScript:** Wykorzystałem czysty JS do pisania skryptów w zakładkach `Pre-request Script` oraz `Tests`.
* **Automatyzacja:** Skrypty automatycznie weryfikują statusy odpowiedzi (200 OK), schematy JSON oraz czasy odpowiedzi.
* **Dynamiczne dane:** Użyłem zmiennych i skryptów do przekazywania danych między różnymi zapytaniami API.

## Jak uruchomić?
Pobierz plik `.json` i zaimportuj go do swojej aplikacji Postman.
